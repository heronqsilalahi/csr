# Generate CSR SSL

## Instalation
Download installer:
```shell
wget https://raw.githubusercontent.com/heronqsilalahi/csr/main/install.sh
```
Allow executable:
```shell
chmod +x install.sh
```
Install scipt:
```shell
./install.sh
```

## Usage
Run command to generate CSR file:
```shell
Generate-CSR
```

## Env File Location
Executable
```shell
/usr/local/bin/Generate-CSR
```

Conf, CSR & Private Key
```shell
/var/openssl/
```
